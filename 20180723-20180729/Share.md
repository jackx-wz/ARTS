# 文档内容搜索 ag

```
以前使用 perl 的时候，一直用的 ack 来搜索文件内容，还有个类似功能的 ag.都是直接开箱即用，个人感觉比 grep 好用，能偷懒的时候就偷懒吧。

ack "中文"      //当前文件夹下所有文件搜索中文
ack "中文" /usr/local/src     //在指定目录下搜索中文

参数也类似

ack -w "usr"     //整词匹配
ag -w "usr"      //整词匹配
```
